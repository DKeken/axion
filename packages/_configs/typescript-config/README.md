# @axion-stack/typescript-config

Shared TypeScript configurations for Axion Stack monorepo.

## Usage

### Base Configuration

```json
{
  "extends": "@axion-stack/typescript-config/base"
}
```

### Next.js Configuration

```json
{
  "extends": "@axion-stack/typescript-config/nextjs"
}
```

### Strict Configuration

```json
{
  "extends": "@axion-stack/typescript-config/strict"
}
```

### Node.js Configuration

```json
{
  "extends": "@axion-stack/typescript-config/node"
}
```

## Installation

This package is part of the monorepo and should be installed as a workspace dependency:

```bash
bun add -D @axion-stack/typescript-config
```
