# Документация для LLM

Эта директория содержит нормализованные документы для использования с LLM (Large Language Models).

## Структура

- `00-ARCHITECTURE.md` — Полная архитектура системы (главный документ)
- `01-TECHNICAL_DETAILS.md` — Технические детали реализации
- `02-CONTRACTS_AND_VALIDATION.md` — Система контрактов и валидации
- `03-RUNNER_AGENT.md` — Архитектура Axion Runner Agent

## Использование

Эти документы оптимизированы для:

- Контекста LLM (убраны лишние форматирования, эмодзи)
- Структурированного поиска
- Автоматической обработки

## Обновление

Документы создаются из исходных файлов в `docs/` директории. При изменении исходных документов, обновите соответствующие файлы здесь.
